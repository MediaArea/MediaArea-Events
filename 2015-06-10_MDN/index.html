What is MediaInfo?

MediaInfo is a convenient unified display of the most relevant technical and tag data for video and audio files.

MediaInfo
is versatile.


Containers/wrappers


  • SMPTE containers (MXF/DCP/IMF)
  • GXF, LXF
  • P2, XDCAM
  • MPEG streaming containers (ts/m2ts/mpg/vob)
  • MPEG file container (mp4)
  • Quicktime (qt/mov)
  • Matroska (mkv/mka/mks)
  • RIFF (avi/wav/w64)
  • Apple/SGI (aiff/aifc)
  • Windows Media (asf/wma/wmv)
  • ...

MediaInfo
is open source.


License


  • BSD-2-Clause license
  • Code source available for everyone
  • Contacting developer not mandatory
  • Not tied to a company

* Simple license (BSD like): Redistribution and use in source and binary forms, with or without modification, are permitted.
means: just cite my name (commercial / proprietary products are welcome)
* You want to improve it with a super feature on your own? Do it!
Contacting the main developer is not mandatory.
You contract him only if his offer suits your needs, else you can contract anyone else.
You are not tied to a company. The company is going in the wrong direction? You fork.

Business model


  • Free: Software & developed features
  • Not free: Closed source versions
  • Not free: Bug correction
  • Not free: New features
  • Not free: Support/forensics

Exception about developed features: some people want to pay for a closed source version of a feature. You can sponsor for making it open source

Automation (3)


File Information Tool Set (FITS)

  • Developed by Harvard University Library Digital Repository Service
  • Identifies and validates file formats, extracts metadata embedded within files, and outputs the metadata in various formats (including EBUCore)
  • Manages the output from several other open source tools
  • Will include MediaInfo as one of the inputs

FITS will include MediaInfo in 2016.
